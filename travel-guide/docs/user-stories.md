# Travel Guide Software — User Stories

---

## User Story nº1
**Content:**  
I am a true traveler, I always try to visit all the important places in the city.

**Summary:**  
A traveler who wants to visit all major landmarks and must-see locations.

**Features Needed:**
- List of important landmarks per city
- “Must-see” recommendations
- Map integration with key locations
- Suggested itinerary for top attractions

**Acceptance Criteria:**
- User can view a list of major attractions in a city
- User can see these locations on a map
- User can generate a basic itinerary including top landmarks

---

## User Story nº2
**Content:**  
I am a sportsmen emeritus, but I always look for places where I can do some sports during my travel or planning visits to the places connected with a sport like museums, stadiums, etc.

**Summary:**  
A sports enthusiast looking for sports activities and sport-related places.

**Features Needed:**
- Search/filter for sports facilities (gyms, stadiums, parks)
- Listings of sports-related attractions (museums, arenas)
- Activity suggestions based on sports

**Acceptance Criteria:**
- User can find sports venues in a city
- User can discover sport-related attractions
- User can filter activities by sport type

---

## User Story nº3
**Content:**  
We are travelling a lot and want to see the most interesting places in each city. It is always difficult to plan a short trip and decide which places we need to visit.

**Summary:**  
Frequent travelers who need help planning efficient short trips.

**Features Needed:**
- Smart itinerary generator for short stays
- Ranking of “most interesting” places
- Time-based trip planning (e.g., 1-day, 2-day trips)

**Acceptance Criteria:**
- User can generate a short-trip itinerary
- Places are ranked by relevance/popularity
- User can customize itinerary duration

---

## User Story nº4
**Content:**  
We love visiting a city where we can learn about the city's history and culture without rushing from one place to another.  
We appreciate hotels that offer comfortable accommodations with elevators and easy access to key attractions.

**Summary:**  
Travelers interested in culture, comfort, and accessibility.

**Features Needed:**
- Cultural and historical place recommendations
- Relaxed/slow-paced itinerary options
- Hotel search with accessibility filters (elevators, proximity)

**Acceptance Criteria:**
- User can find cultural and historical attractions
- User can choose a “relaxed pace” itinerary
- User can filter hotels by accessibility features and location

---

## User Story nº5
**Content:**  
I love art … for me, a perfect city visit starts with knowing where the best galleries and street art are. I love discovering both famous museums and hidden artistic gems.

**Summary:**  
An art lover seeking both famous and hidden artistic locations.

**Features Needed:**
- Listings of museums, galleries, and street art spots
- Highlight “hidden gems”
- Art-focused city guides

**Acceptance Criteria:**
- User can browse art-related places
- User can discover lesser-known art spots
- User can filter by type (museum, gallery, street art)

---

## User Story nº6
**Content:**  
For us, the perfect city trip means exploring on our bikes. We always look for cities with dedicated cycling lanes and safe routes.  
Biking lets us see so much more than just the main tourist spots, we can travel faster and see more places.  
We love discovering hidden streets, charming neighborhoods, and scenic parks at our own pace.

**Summary:**  
Cycling travelers looking for bike-friendly routes and exploration.

**Features Needed:**
- Cycling routes and bike lanes map
- Route planning optimized for bikes
- Discovery of neighborhoods and parks

**Acceptance Criteria:**
- User can view bike-friendly routes on a map
- User can generate cycling itineraries
- User can explore points of interest along routes

---

## User Story nº7
**Content:**  
When I plan a city visit with my kids, I look for places that are both fun and educational.  
Museums with interactive exhibits or science centers are always a hit!  
A good city for traveling with children needs plenty of green spaces. Playgrounds, parks, and areas where they can run around are essential for happy little explorers.

**Summary:**  
A parent looking for family-friendly, fun, and educational activities.

**Features Needed:**
- Family-friendly activity recommendations
- Filters for kid-friendly museums and attractions
- Listings of parks and playgrounds

**Acceptance Criteria:**
- User can find activities suitable for children
- User can filter for interactive/educational places
- User can locate parks and green spaces

# Recap of Features

## Core Features to Implement:
- Landmark and “must-see” recommendation system
- Smart itinerary generator (including short and relaxed trips)
- Interactive map integration
- Filtering system (by interest: sports, art, family, culture, etc.)
- Hotel search with accessibility options
- Hidden gems recommendation system
- Sports facilities and attractions database
- Cycling routes and bike-friendly navigation
- Family-friendly activity tagging
- Park and green space listings
